# Portfolio (Vite + React + Three + Tailwind)

Quick start instructions to run the project locally.

Installation

```bash
npm install
```

Tailwind setup (already added config files here). If you want to re-run the initializer:

```bash
npx tailwindcss init -p
```

Run dev server

```bash
npm run dev
```

Build

```bash
npm run build
```

Preview build

```bash
npm run preview
```

Files of interest:

- `index.html`
- `src/main.jsx`
- `src/portfolio.jsx`
- `src/index.css`
- `tailwind.config.cjs`
